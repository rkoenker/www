Getting Mathematica to run in Ragnar

Ragnar has a site license for Mathematica version 4 that should enable all Ragnar account holders to use it at any time. However, at the moment only two concurrent sessions of mathematica can be run. If you cannot lauch the program, it is most likely because two other people are already using it. An easy way to check on this is to use the ps unix command and look for other instances of mathematica.

Please contact Daniel Morillo for questions related to this.
