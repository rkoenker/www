Regression quantiles function for SPLUS

This page contains the basic files to install the regression quantile function in Splus for Windows. You will need the following files: readme.txt, rqhelp.txt, rq.s and rq.obj The first three files are in readable format and can be downloaded separately. rq.zip is a compressed file containing the four files.

Note that functions compiled for Splus version 4.0 are NOT compatible with Splus 3.4 and below (and viceversa).

The following are needed for SPLUS 3.4 and below

readme.txt
rqhelp.txt
rq.s

rq.zip 		Instructions to install the rq() function in Splus for Windows (ASCII).
Help file. (ASCII)
Splus quantile regression function (ASCII).
