\noindent \textbf{observations:} 
\[
(y_t, x_t), \quad\quad      t = 1,\ldots, n 
\]
\[
y_t \ge 0, \quad y_t = 1\times 1, \quad  x_t = k\times 1
\]

\bigskip

\noindent \textbf{model:}  censored regression model or ``Tobit''
\[
y_t = \max \{0, x_t' \beta_0 + u_t\}, \quad   t = 1,\ldots, n. 
\]

\bigskip

\noindent \textbf{problem:}  estimate the population parameter 
$\beta$ using the LAD estimator
${\hat \beta}$.  (Powell, 1984)
Since the median of median of $y_t$ is 
\[
m(y_t) = \max\{0, x_t' \beta_0\}
\] 
the censored LAD estimator is obtained as the solution to the problem:
\[
\min_\beta\sum_{t=1} ^ n | y_t - \max \{0, x_t'\beta\} |.
\]

\bigskip

\noindent \textbf{FOC:}  alternatively, the estimator ${\hat \beta}$ 
can be obtained as the root of
\[
S_x (\beta) = n^{-\frac {1}{2}} \sum_{t=1}^n I(x_t' \beta > 0)\{I (y_t - x_t'
\beta\le 0) - \frac {1}{2}\} x_t
\]
Since $E\{S_x (\beta_0)\} = 0, S_x (\beta)$ can serve as an estimating
equation.

\noindent \textbf{Resampling Method:} \\
A new resampling method has been proposed by
Parzen, Wei and Ying (Biometrika, 1994) recently, when the random vector
$S_x(\beta_0)$ is exactly or asymptotically pivotal.  That is, the exact or
limiting distribution is completely known or can be estimated consistently.

We apply this resampling method to the censored LAD estimator proposed by
Powell (1984).  In this case, the conditional distibution (given $x$) of
$S_x(\beta_0)$ can be exactly generated by a weighted sum of independent and
centered Bernoulli variables with probability of ``success'' $\frac{1}{2}$:
\[
U = n^{-\frac{1}{2}} \sum _{t=1}^n I(x_t' \beta_0 > 0) \{\xi_t -
\frac{1}{2}\} x_t.
\]
To explain in more detail note that, when $I(x_t'\beta_0 > 0) = 1$, it follows
that $x_t' \beta = \max\{0, x_t'\beta_0\} = m(y_t|x_t)$ and therefore $I(y_t -
x_t'\beta_0 \le 0)$ is a Bernoulli variate.  On the other hand, when $I(x_t'
\beta_0 > 0) = 0$ the corresponding summand of $S_x(\beta_0)$ is zero.  Thus,
the summation in $U$ runs over the observations for which $x_t'\beta_0 > 0$ 
holds:
\[
U = n^{-\frac{1}{2}} \sum_{t:x_t' \beta_0>0} \{\xi_t - \frac {1}{2}\} x_t.
\]
The distribution of $U$ can be consistently estimated by
\[
{\hat U} = n^{-\frac{1}{2}} \sum_{t=1}^n I(x_t'{\hat \beta} > 0) \{\xi_t -
\frac {1}{2} \} x_t.
\]
Under conditions, for large $n$, the distribution of $(\hat \beta - \beta_0)$
can be approximated by the conditional distribution of $({\tilde \beta} -
\beta_U )$.  To generate the distribution of $\beta_U$ empirically we need to
be able to solve the equation
\[
S_x(\beta) = {\hat u}
\]
for a given realization of ${\hat u}$ of ${\hat U}$.  To this end, create an
extra data point $(y_{n+1}, x_{n+1})$ with $y_{n+1}$ extremely large so that
$I(y_{n+1} - x_{t+1}' \beta \le 0)$ becomes always 0.  For a given ${\hat
u}$, we generate $x_{n+1} = 2n^{\frac {1}{2}} {\hat u}$.  Now, note that 
\[
n^{-\frac {1}{2}} \sum_{t=1}^{n+1} I(x_t' \beta > 0) \{ I(y_t - x_t'
\beta \le 0 )  -\frac{1}{2} \} x_t = 0
\] 
is equivalent to
\[
n^{-\frac {1}{2}} \sum_{t=1}^n I(x_t' \beta > 0 ) \{ I(y_t - x_t' \beta
\le 0 )  -\frac {1}{2} \} x_t 
= I(x_{n+1}' \beta > 0 ) {\hat u}
\]
while we would like the right-hand side of last equality to be ${\hat u}$. 
%% That is, we would like to have $x_{n+1}' \beta > 0$ always.  
Therefore
\[
S(\beta) = {\hat u} 
\]
is equivalent to 
\[
n^{-\frac {1}{2}} \sum_{t=1}^n I(x_t' \beta > 0) 
\{ I( y_t - x_t' \beta \le 0) {-\frac {1}{2}} \} x_t
+  n^{-\frac {1}{2}} \{I( y_{n+1} - x_{n+1}' \beta \le 0) {-\frac {1}{2}}
\} x_{n+1} = 0
\]
which in practice can be accomplished by minimizing the objective function:
\[
\min_\beta \left\{ \sum_{t=1}^n | y_t - \max \{0, x_t' \beta \} | + 
| y_{n+1} - x_{n+1} ' \beta | \right\}
\]
%% That is, we have incorporate the restriction 
%% $x_{n+1}' \beta > 0$ in the
%% objective function by setting 
%% $\max \{0, x_{n+1} ' \beta \} = x_{n+1} ' \beta$.

\bigskip

Hopefully, this optimization program can be carried out with the algorithm for
nonlinear regression quantiles developed recently by Koenker and Park 
(J.\ of Econometrics, 1995).
