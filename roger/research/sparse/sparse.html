SparseM: Sparse Linear Algebra for R


This is an archive page for software and text for package for sparse linear algebra in R. A brief description of the package with some examples of its functionality (a vignette, in the terminology of R) is available in pdf here, and all of the source code and documentation is available on CRAN.
This package is joint work by Roger Koenker and Pin Ng. Comments on this are, of course, always welcome, and should be directed to roger@ysidro.econ.uiuc.edu or Pin.Ng@nau.edu
