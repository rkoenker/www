UCL Master Class


This page provides links to files containing the slides and other materials for the class. There is also a short overview of quantile regression terrain with some references, and a set of exercises designed for the course, and documentation of the quantile regression software I have written for R. For further reference to materials on quantile regression you might consult my main webpage.

The lecture slides can be printed 4 to a page using the acroread or some other facility. The lecture slides are in a preliminary form suitable for printing and are consequently marred by some mildly annoying small black rectangles that are an artifact of the ppower4 formatting. As mentioned in class the slides were prepared with pdflatex and postprocessed with the public domain java program ppower4. For a useful example of the use of the latter program see here, and go up one level to see the files that were used to create the final product.

There are three data sets provided in conjunction with the exercises. Two are in standard ascii format and can be read into R using read.table as described in the exercises, the third, the socalled motorcycle data is stored in R-format and can be read using the command: source("motorcycle.R"). The software for the qrisk problem is also made available in the form of an R dataset that can be read with the source command. In addition I've included the Melbourne temperature data, and the code to make the density plots. Note that the latter uses an implementation in quantreg of the adaptive kernel density estimation method of Silverman(1984) called akj().
