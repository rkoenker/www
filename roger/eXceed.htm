X Display at CCSO Labs


Several of you are interested in accessing our servers (*.econ.uiuc.edu)
with the regular X-interface. While the Econometrics Lab is
temporarily unavailable you can do it at the CCSO Labs.


How To X-Display:


1) Network Services Folder: Run eXceed

2) In the taskbar there's a new icon -- right click and select
   a) Tools
b) Client Startup
c) New...

3) In the new window please provide the following information:

  Start Method: RLOGIN(TCP/IP)
 Program Type: X Windows
 User ID: your_econ_id
 Password: xxx
 Host: *.econ.uiuc.edu (where *=s2, galton,...)
 Host Type: Leave blank
 Command: xterm -sl 1000 -sb -display @d &
 Prompt: Leave blank

4) Click on Run (taskbar)

5) A new window pops up. Business as usual, for example, start Matlab or Splus.
